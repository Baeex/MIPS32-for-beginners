# MIPS32-for-beginners
This repository contains a small collection of MIPS32 assembly programs written to run on **QtSpim**.  
The programs cover basic input/output, arithmetic, conditionals, loops, and arrays.

They are intended as simple, standalone programs that demonstrate fundamental concepts of MIPS assembly programming.
